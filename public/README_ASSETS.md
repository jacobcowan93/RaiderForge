This project expects game assets (images, logo, video) to be available under `public/images`.

- Move or symlink the existing `assets/` folder into `public/images` so Next.js can serve them.
- Example:

  mv ../assets public/images

- Required assets used by the scaffold:
  - `public/images/ARC_Home.mp4` (hero background video)
  - `public/images/logo.png` (site logo) — replace with your logo filename
